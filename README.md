DSA Instructor Chatbot

A simple chatbot that helps users learn Data Structures and Algorithms.

It answers DSA questions politely, and if the user asks something unrelated, it replies in a fun and slightly rude way.

Features

Chat interface (like modern AI chat apps)

Remembers recent chat history

Uses Google Gemini API

User enters their own API key

Local storage support

Typing indicator

How to Use

Open the website

Click the API Key section

Get a key from:

https://aistudio.google.com/app/apikey

Paste the key in the input box

Start chatting

Tech Used

HTML

CSS

JavaScript

Gemini API
